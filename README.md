# Biological Systems Modeling – MATLAB Projects

This repository contains MATLAB projects developed for the course *Biological Systems Modeling*, focused on the mathematical and computational modeling of biological and physiological systems.

## Topics Covered
- Membrane electrical models
- Systemic circulation
- Cardiovascular control mechanisms
- Hemodialysis simulation
- Respiratory mechanics
- Assisted ventilation
- Ventilation control
- Oxygen and carbon dioxide exchange
- Chaos models (Lorenz and Rossler systems)
- Hodgkin-Huxley neural model
